# TikTok Trends Reference

Current and evergreen TikTok trends and formats for content creation.

## Evergreen Formats (Always Work)

### 1. "POV" Format
- POV: You finally found...
- POV: You're the friend who...
- POV: It's [year] and...
- **Why it works:** Creates immediate immersion and relatability

### 2. "Stop Doing X" Format
- Stop doing [common mistake]
- Stop buying [product]
- Stop [action] if you want [result]
- **Why it works:** Pattern interrupt + authority positioning

### 3. "Things I Wish I Knew" Format
- Things I wish I knew at [age]
- Things I wish I knew before [event]
- Things nobody tells you about [topic]
- **Why it works:** Provides value + curiosity gap

### 4. "Day in the Life" Format
- Day in the life of a [profession]
- Morning routine of [type of person]
- What I spend in a day as [identity]
- **Why it works:** Satisfies curiosity + lifestyle aspiration

### 5. "Transformation" Format
- Before/After reveals
- Glow up journeys
- Progress tracking over time
- **Why it works:** Visual payoff + emotional journey

## Audio Trends

### How to Find Trending Audio
1. TikTok Creative Center (ads.tiktok.com/business/creativecenter)
2. Scroll FYP and note repeated sounds
3. Check TikTok's "Trending" tab
4. Follow creator accounts that track trends

### Using Audio Effectively
- Jump on trends early (first 2-3 days)
- Put your own spin on trending sounds
- Use audio that matches your niche
- Don't force trends that don't fit your brand

## Video Structure Best Practices

### The 3-Second Hook
- Start with the most interesting frame
- Use text overlay with curiosity gap
- Show the result before the process
- Ask a question that demands an answer

### Retention Techniques
- Change scene every 3-5 seconds
- Use jump cuts to maintain pace
- Add text overlays for key points
- Include pattern interrupts (sounds, zooms, cuts)

### The Ending
- Clear call-to-action
- Tease next video
- Ask a question to drive comments
- Use "follow for part 2" for series content

|--------------|-------------|------------|
| Quick tips | 15-30 sec | 45 sec |
| Tutorials | 45-90 sec | 3 min |
| Story times | 60-180 sec | 3 min |
| Entertainment | 15-60 sec | 90 sec |
| Educational | 30-90 sec | 2 min |

## Posting Frequency Guidelines

### Growth Phase
- 1-3 videos per day
- Consistent posting times
- Test different content types
- Engage heavily with comments

### Maintenance Phase
- 3-7 videos per week
- Focus on quality over quantity
- Analyze and double down on what works
- Batch create content

## Engagement Tactics

### Comments Strategy
- Reply to every comment in first hour
- Pin a question to drive more comments
- Like and heart comments
- Create video responses to top comments

### Duet/Stitch Opportunities
- React to viral content in your niche
- Add value to popular videos
- Collaborate with similar-sized creators
- Participate in challenges early
